# 🃏 Black Jack Game

A simple and interactive Black Jack game built using HTML, CSS, and JavaScript.

This project was created to practice JavaScript fundamentals like:
- Arrays
- Objects
- Functions
- DOM Manipulation
- Conditional Rendering
- Game Logic

---

## 🚀 Features

✅ Random card generation  
✅ Black Jack win detection  
✅ Lose condition handling  
✅ Dynamic UI updates  
✅ Draw new cards  
✅ Reset / New Game functionality  
✅ Player details display  
✅ Button disabling logic  

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

---

## 🎮 How to Play

1. Click **Start Game**
2. A random card will be drawn
3. Click **New Card** to draw more cards
4. Reach exactly **21** to win 🏆
5. If the sum goes above **21**, you lose ❌
6. Click **New Game** to restart
